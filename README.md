# 🌸 Pinky Promise Nail Club — App de Clientas

App PWA para que las clientas puedan reservar citas online.

## 📱 Instalar en iPhone
1. Abre Safari en tu iPhone
2. Ve a https://eiderfreile.github.io/PinkyPromiseClientes/
3. Pulsa Compartir → Añadir a pantalla de inicio

## ✨ Funcionalidades
- Ver carta de servicios con tiempos y precios
- Reservar cita eligiendo servicio, subservicios, fecha y hora
- Ver huecos disponibles según tu agenda
- Ver y cancelar sus propias reservas

## 🔗 Conexión con tu app de gestión
- Las reservas llegan directamente a tu pestaña Agenda → Pendientes
- Las clientas nuevas pueden ser añadidas a tu lista desde la reserva
- Toda la disponibilidad se sincroniza en tiempo real con Firebase
